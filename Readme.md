## Instalação e execução

  1. Faça um clone desse repositório;
  2. Entre na pasta rodando `cd masterclass-sejadev`;
  3. Execute o comando `npm run dev` no terminal.
  4. Abra no navegador a seguinte URL: `http://localhost:3000/`.
